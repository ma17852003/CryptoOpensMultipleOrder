# Crypto 一鍵多幣種同步開倉助手 (MEXC)

這是一個專為量化交易與高頻交易者設計的 **MEXC 合約同步操作面板**。
當遇到大盤（如 BTC、ETH、SOL、SUI 等）同步大漲或大跌時，您可以透過此系統在同一毫秒內一鍵同時開啟多個幣種的合約倉位，或是「一鍵閃電全平倉」，絕不錯過最佳進出場點位。

## 🌟 核心功能
- **一鍵多幣同時開倉**：自訂所有主流幣種，同時針對多個幣種發出市價市價多/空單。
- **嚴格成本與槓桿控制**：
  - 我說 1u 就是 1u：輸入多少 USDT 本金，系統自動精算最貼近的合約張數。如果本金不足以購買 1 張，系統將**嚴格報錯並提示所需最小金額**，絕不多花您一毛錢。
  - 嚴格全倉鎖定：系統與 MEXC API 溝通時，強制死鎖於「全倉模式」，並嚴格區分多空屬性，確保槓桿精準套用。
- **本地端極致安全**：您的 API Key 與設定參數皆自動記憶在**您本機的隱藏設定檔** (`backend/config.json`) 中，絕不上傳至任何雲端或 Github，保障資產 100% 安全。
- **完全 Docker 化**：內建前後端隔離環境，只需一行指令即可一鍵啟動。

## 🚀 快速安裝與啟動教學 (Docker)

本專案強烈建議使用 Docker 運行，這能確保您不需要煩惱任何 Node.js 版本或依賴衝突的問題。

### 第一步：安裝 Docker (若您的電腦還沒安裝)
1. 前往 Docker 官方網站下載 **Docker Desktop**:
   👉 [點此下載 Docker Desktop (Windows/Mac/Linux)](https://www.docker.com/products/docker-desktop/)
2. 下載後執行安裝檔，並一直按「Next」直到完成。
3. 安裝完成後，請**打開 Docker Desktop 軟體**，並確保它在背景運行（看得到一艘載著貨櫃的小鯨魚圖示為綠色狀態）。

### 第二步：啟動本系統
打開您的終端機 (Terminal / PowerShell)，進入到本專案的資料夾，輸入以下指令：

```bash
docker-compose up -d --build
```

這行指令會自動幫您打包好前端與後端環境。
（如果未來有更新代碼，也只需要重新執行這行指令即可套用更新）。

### 第三步：開始使用
當 Docker 跑完進度條並顯示 `Started` 後，打開您的瀏覽器，輸入：
👉 **http://localhost:5173**

您就可以開始綁定 MEXC 的 API Key 並使用了！

## 🔧 系統架構
- **前端**: React + Vite + TypeScript (玻璃擬物化 UI)
- **後端**: Node.js + Express + CCXT (負責精準繞過交易所速率限制與下單運算)
- **容器**: Docker + Docker-compose + Nginx

## ⚠️ 安全性與免責聲明
- 請確保您在 MEXC 申請 API Key 時，**僅開啟「讀取」與「合約交易」權限**，絕對不要開啟「提現」權限。
- 加密貨幣合約交易具有極高風險，本開源工具僅提供發送 API 之便利性，不對任何交易虧損負責。
